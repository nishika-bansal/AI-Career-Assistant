# 🤖 AI Career Assistant – Smart Job Matching System

## 📌 Overview
AI Career Assistant is a Flask-based web application that helps users find jobs based on skills using smart keyword matching logic.

---

## 🚀 Features
- Skill-based job search
- Smart keyword mapping
- Modern dashboard UI
- Save job simulation
- Email generation concept UI

---

## 🏗️ Tech Stack
- Python
- Flask
- HTML
- CSS
- Jinja2
